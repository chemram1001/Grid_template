Arduino C Programming
Algorithms

Since the Arduino is a microcontroller, computational resources are very limited. This requires that Arduino programmers be mindful of the resources that their code uses. To begin, we Look at the different data types supported by the Arduino C language.

Note: 1 byte of data is equal to 8 bits, and 1 bit of data is either a 1 or a 0. Also, 1 byte of data has binary values thus, 0 would be shown as 00000000 as 1 byte of 8 bits. Similarily, the number 1 in binary is shown as 00000001 and the number 2 is 00000010, etc

From the image above we can see that the boolean data type occupies 1 byte of information and as such has possible values of 0 or 1. The next data type is the char which has 1 byte of data and has values ranging from -127 to 128.
